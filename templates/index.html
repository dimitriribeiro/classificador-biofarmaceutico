Seja Bem-vindo a API de Classificação Biofarmacêutica!

Essa é uma API produzida para ajudar os pesquisadores da área de ciências da saúde, em especial, os da área farmacêutica. Muitas vezes, quando estamos pesquisando sobre insumos farmacêuticos ativos temos que encontrar uma informação muito importante que é a classificação biofarmacêutica.

Essa informação nos diz dois dados importantes sobre as substâncias ativas que podem ser vinculados em medicamentos. A solubilidade e a permeabilidade. A primeira nos diz sobre a capacidade da substância se dissolver em água e a segunda se trata da capacidade do fármaco ultrapassar as membranas células do corpo.

Contudo, essas informações de vital importância para produção e pesquisa de medicamentos ainda estão muito dispersas na web. A ideia dessa API e resolver essa problemática, trazendo para os pesquisadores que a utilizarem a facilidade de acesso da classificação biofarmacêutica. E além disso, em breve esse site será aberto para contribuições de pesquisadores no Brasil inteiro.

O que é uma API?

API significa Interface de Programação de Aplicativos (do inglês Application Programming Interface). É um conjunto de regras e protocolos que permite a comunicação entre diferentes softwares.

Onde iniciou a base que alimenta essa API?

As informações que alimentaram a base de dados dessa API tem como fonte a última edição da farmacopeia brasileira.
